# 🏃 马拉松跑鞋品牌数据分析平台

深度分析乔丹品牌及国产/国际品牌在马拉松赛场上的地位变化。

## 📊 功能模块

### 1. 🏠 总览
- 核心数据指标展示
- 乔丹品牌排名趋势
- 国产vs国际品牌占比趋势
- 关键洞察总结

### 2. 👟 乔丹专题
- 乔丹品牌核心指标（最佳/最差排名、平均份额）
- 排名变化趋势图
- 市场份额变化图
- 各赛事表现热力图
- 智能分析报告

### 3. 🌏 国产vs国际
- 市场份额变化趋势
- TOP10品牌数量变化
- 代表品牌排名趋势
- 崛起分析报告

### 4. ⚖️ 品牌对比
- 自由选择品牌对比（最多5个）
- 排名/份额对比图表
- 综合实力雷达图
- **智能分析报告自动生成**

### 5. 📊 数据浏览
- 完整数据表格
- 多维度筛选
- CSV导出功能

## 🚀 本地运行

```bash
# 安装依赖
pip install -r requirements.txt

# 运行应用
streamlit run app.py
```

## ☁️ 部署到Streamlit Cloud

1. 将项目推送到GitHub仓库
2. 访问 [Streamlit Cloud](https://share.streamlit.io/)
3. 点击 "New app"
4. 选择GitHub仓库和分支
5. 设置主文件为 `app.py`
6. 点击 "Deploy"

部署完成后即可获得公开链接分享给任何人！

## 📌 数据来源

数据来源：悦跑圈等平台统计的马拉松跑鞋穿着率排行榜

## 🎯 分析目标

1. **乔丹品牌分析**：追踪乔丹跑鞋在马拉松赛场上的江湖地位变化
2. **国产vs国际**：分析国产品牌在马拉松赛场上的崛起之路
3. **品牌对比**：支持任意品牌的深度对比分析
